# mpd-lrc
This was for learning purposes (go stuff).

A useless cli to show current song lyrics, using mpd

## Usage

Just create a the config file `mpdLrc.toml` at your `XDG_HOME_CONFIG` or `~/.config`.
Build, and that's it.
